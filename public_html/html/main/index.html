간결한 코드

복잡한 설정 없이 빠르게 개발을 시작할 수 있습니다. JSP와 HTML이 완전히 분리되어 유지보수가 쉽습니다.

보안

SQL Injection, XSS 공격을 자동으로 방어합니다. PreparedStatement와 자동 escape 처리를 지원합니다.

DataObject, Form, Page 클래스로 CRUD 작업을 빠르게 구현할 수 있습니다.

  • Bootstrap 5 반응형 디자인
  • 페이징 및 검색 기능
